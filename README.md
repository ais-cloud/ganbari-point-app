# 📚 がんばりポイント

**幼児・小学生の勉強習慣を作るための、親子で簡単ポイント管理アプリです。**

勉強や習い事をポイントにして、貯めたポイントをご褒美と交換。
「やったらいいことがある」の仕組みで、子どもの「続ける力」を育てます。

---

## 🔗 デモを触ってみる

**[👉 こちらから実際に使えます](https://ais-cloud.github.io/point-app/)**

## ✨ こんなことができます

- 📝 **ためる** ─ プリント1枚◯ポイント、ドリル1ページ◯ポイントと家庭で自由に設定
- 💰 **つかう** ─ YouTube視聴時間、Switch時間などのご褒美と交換
- 📅 **見える化** ─ カレンダーに毎日の頑張りを銅・銀・金メダルで表示
- 🔒 **親が見守る** ─ PINコードで、ご褒美の使用は親が最終承認
- ⚙️ **自由にカスタマイズ** ─ 学習項目もご褒美も家庭ごとに設定可能

## 🎯 こんな方におすすめ

- 子どもの勉強習慣を楽しく作りたい
- 「やらせる」ではなく「やりたくなる」仕組みがほしい
- YouTubeやゲームの時間管理に悩んでいる
- 既存アプリの広告やサブスクに抵抗がある
- 家庭に合わせて柔軟に使えるシンプルなツールを探している

## 🛠 使っている技術

- HTML / CSS / JavaScript(フレームワークなし)
- ブラウザのローカルストレージでデータ保存
- GitHub Pagesで公開
